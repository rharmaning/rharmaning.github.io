_A simple Jekyll + GitHub Pages powered resume template._

Big thanks to [Matt Skone](https://github.com/mattskone) (and by extension [Joel Glovier's project](https://github.com/jglovier/resume-template)) for this template.

## Local workflow

- `make install` installs and verifies the pinned Ruby/Bundler/Jekyll setup
- `make serve` starts the local resume site at `http://127.0.0.1:4000`
- `make check` checks formatting for `_layouts/resume.html`
- `make resume-format` formats `_layouts/resume.html`
